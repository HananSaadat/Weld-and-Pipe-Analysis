Title:


Image Processing for Structural Analysis of Welds and Pipes

Description:

This project focuses on the detection and analysis of structural features in weld and pipe images using image processing techniques. The goal is to identify and visualize potential defects and irregularities to enhance the reliability and safety of welded structures. This project utilizes OpenCV, TensorFlow, and Keras for processing and analyzing X-ray images of welds and pipes, showcasing the effectiveness of automated image analysis in industrial applications.
